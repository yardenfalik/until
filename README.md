## 🚀 Technologies
- React + TypeScript
- CSS Modules
- Vite
- PWA (Progressive Web App)

## 🛠️ Installation
1. Open the app in Safari or Chrome on your phone ([in this link](https://until-app.vercel.app/)).
2. Tap "Share" → "Add to Home Screen".
3. That's it! Use it like any installed app.
